The current UI looks AI-generated, generic, and template-like. I want a complete design refinement pass that makes AthleteAI feel like a premium consumer sports app that could realistically rank in the App Store.

Do NOT simply add more gradients, shadows, or colors. Focus on thoughtful design, hierarchy, spacing, consistency, and polish.

Primary Goal:

Make the app feel designed by a top-tier product designer, not generated by AI.

Apps to take inspiration from:

* Linear
* Arc Browser
* WHOOP
* Strava
* Notion Calendar
* Apple Fitness
* Nike Run Club
* Duolingo (for engagement, not visual style)

Things that currently make the app feel AI-generated:

* Generic card layouts
* Excessive use of equally-sized boxes
* Too many sections competing for attention
* Poor visual hierarchy
* Large walls of text
* Inconsistent spacing
* Generic gradients
* Repetitive UI patterns
* Lack of personality
* Too many borders
* Screens that feel cluttered

Design System Improvements:

Create a consistent design language:

* Consistent corner radii
* Consistent spacing scale
* Consistent typography system
* Consistent icon system
* Consistent color system

Every screen should feel like it belongs to the same product.

Typography:

Completely improve typography hierarchy.

Create clear distinctions between:

* Page titles
* Section headers
* Card titles
* Body text
* Supporting text
* Data values

Reduce visual noise.

Use larger headings and less text.

Make important information immediately obvious.

Spacing:

Increase whitespace significantly.

The UI currently feels cramped.

Add more breathing room:

* Between sections
* Between cards
* Around content
* Around buttons

Nothing should feel squeezed together.

Analysis Screen Redesign:

This is the most important screen.

Current issue:
The analysis feels like a report generated by AI.

Desired feeling:
A sports coach reviewing your performance.

Redesign the page so users instantly see:

1. Overall Result
2. Biggest Strength
3. Biggest Weakness
4. Next Action

Everything else should be secondary.

Reduce cognitive load.

Users should understand their analysis within 5 seconds.

Cards:

Stop using generic stacked cards everywhere.

Introduce:

* Hero sections
* Highlight panels
* Visual callouts
* Interactive components
* Timeline elements
* Progress visuals

Create more variety in layouts.

Color Usage:

Use color intentionally.

Do not assign random colors to everything.

Reserve color for:

* Important metrics
* Warnings
* Success states
* Progress indicators
* Interactive elements

Most of the interface should rely on strong layout and typography rather than color alone.

Icons:

Replace generic-looking icons.

Use a cohesive icon set throughout the app.

Icons should support understanding rather than decorate the interface.

Microinteractions:

Add subtle premium interactions:

* Smooth page transitions
* Better loading states
* Hover effects where applicable
* Press animations
* Progress animations
* Skeleton loading states

Nothing should feel abrupt.

Navigation:

Improve navigation clarity.

Users should always know:

* Where they are
* What they should do next
* How to get back

Reduce friction.

Empty States:

Replace generic empty states.

Make them feel intentional and branded.

Loading States:

Replace generic spinners.

Create AthleteAI-specific loading experiences.

Examples:

* Tracking movement
* Mapping joints
* Analyzing technique
* Building coaching insights

Component Audit:

Review every screen and identify:

* Components that look generic
* Components that look repetitive
* Components that look unfinished
* Components that look auto-generated

Replace them with more polished alternatives.

Consistency Audit:

Review:

* Padding
* Margins
* Font sizes
* Border radii
* Shadows
* Button styles
* Card styles
* Colors

Standardize everything.

Final Goal:

The finished product should feel like a venture-backed sports technology startup built by an experienced design team.

A user should never look at the interface and think:
“This looks AI-generated.”

Instead they should think:
“This feels like a real product that people spent months designing.”

After completing the redesign:

* Identify remaining weak areas.
* List screens that still need refinement.
* Suggest additional improvements that would bring the app closer to the quality of WHOOP, Strava, or Linear.
